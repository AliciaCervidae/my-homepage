# Alicia 的个人主页


## 介绍

这是一个个人主页网站，由 Alicia（鹿鹿🦌）创建，用于展示个人信息、作品和技能。网站包含首页和关于页两个主要页面，设计简洁美观，响应式布局适配各种设备。

### 网站功能

- **首页**：包含 Hero 区域、个人信息展示、作品展示和页脚
- **关于页**：详细介绍个人背景、技术栈、兴趣爱好和联系方式
- **响应式设计**：适配桌面端、平板和移动设备
- **现代化 UI**：使用 Tailwind CSS 和 Radix UI 组件库
- **平滑过渡**：添加了交互动画效果

### 网站目的

- 展示个人技能和项目作品
- 提供个人信息和联系方式
- 作为个人品牌的线上展示平台
- 练习和应用前端技术栈

## 目录结构

```
├── README.md # 说明文档
├── components.json # 组件库配置
├── index.html # 入口文件
├── package.json # 包管理
├── postcss.config.js # postcss 配置
├── public # 静态资源目录
│   ├── favicon.png # 图标
│   └── images # 图片资源
├── src # 源码目录
│   ├── App.tsx # 入口文件
│   ├── components # 组件目录
│   ├── contexts # 上下文目录
│   ├── db # 数据库配置目录
│   ├── hooks # 通用钩子函数目录
│   ├── index.css # 全局样式
│   ├── layout # 布局目录
│   ├── lib # 工具库目录
│   ├── main.tsx # 入口文件
│   ├── routes.tsx # 路由配置
│   ├── pages # 页面目录
│   ├── services  # 数据库交互目录
│   ├── types   # 类型定义目录
├── tsconfig.app.json  # ts 前端配置文件
├── tsconfig.json # ts 配置文件
├── tsconfig.node.json # ts node端配置文件
└── vite.config.ts # vite 配置文件
```

## 技术栈

### 前端
- **框架**：React 18
- **语言**：TypeScript
- **构建工具**：Vite
- **样式**：Tailwind CSS
- **UI 组件库**：Radix UI
- **路由**：React Router
- **状态管理**：React Context API

### 后端
- **数据库**：Supabase

### 其他工具
- **设计工具**：Figma
- **图片处理**：Adobe / Pixelmator Pro
- **绘画工具**：Clip Studio Paint
